# BUILDIN Website

Korean Wellness & Inner Beauty brand website.

## Structure

```
website/
├── index.html          # Main page
├── assets/
│   ├── css/style.css   # Styles
│   ├── js/main.js      # Scripts
│   └── images/         # Images
└── SETUP-GUIDE.md      # Setup guide
```

## Deployment

The website is deployed to GitHub Pages automatically when changes are pushed to `main`/`master` branch via the `deploy-website.yml` workflow.
